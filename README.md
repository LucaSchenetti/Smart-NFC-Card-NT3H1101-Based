📇 NFC Business Card – NT3H1101
📖 Overview

This project is a custom NFC business card based on the NT3H1101 IC.
The card is designed to instantly open my LinkedIn profile when scanned with any NFC-enabled smartphone.

It’s a simple and effective way to share contact information in a modern, contactless way.

⚙️ Features
NFC Type 2 Tag compatible
Instant redirect to LinkedIn profile
No battery required (passive NFC)
Works with most Android and iOS devices
Compact PCB design

🧰 Hardware
NFC IC: NT3H1101
Custom PCB antenna design
Standard ISO14443A compatibility
📡 How It Works

The NFC chip stores an NDEF message containing a URL.
When a smartphone scans the card, it reads the message and automatically opens the LinkedIn profile in the browser.

✏️ Customization

This NFC business card can be easily personalized with your own name and information.
You can update the NDEF message to include your personal LinkedIn profile or any other link you prefer.

🚀 Getting Started
Program the NT3H1101 with your NDEF message
Insert your LinkedIn profile URL
Test with an NFC-enabled smartphone

💡 Use Cases
Digital business cards
Networking events
Smart badges
Contactless sharing
